# TOOLS — Infrastructure Architect

## Skills（能力嫁接）

Skills 是 OpenClaw 已安装的"能力包"（不需要在 workspace 里建 skills 文件夹）。

- 查看可用技能：`openclaw skills list`
- 检查技能依赖是否齐备：`openclaw skills check`

## 技术工具

- 分布式系统：Kubernetes, Docker, etcd, Consul
- 存储：Kafka, Pulsar, HDFS, Ceph, MinIO
- 共识协议：Raft, etcd, Consul
- 监控：Prometheus, Grafana, Jaeger
