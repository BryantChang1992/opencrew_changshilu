# TOOLS — Performance Engineer

## 可用工具

### 性能分析

- pprof: Go CPU/内存分析
- flamegraph: 火焰图生成
- perf: Linux 性能分析
- strace: 系统调用追踪

### 压测工具

- wrk/wrk2: HTTP 压测
- ab: Apache Benchmark
- k6: 现代压测工具
- JMeter: 全功能压测

### 监控/观测

- Prometheus + Grafana
- Jaeger: 分布式追踪
- OpenTelemetry

## 环境配置

- 无特殊环境依赖
- 需要时可用 spawn research 获取工具使用指导
